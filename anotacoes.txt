Iniciar Servidor:

cd desafio
py manage.py runserver

Criar Super Usuario;

py manage.py createsuperuser

Alterar tabela vitrines:

python manage.py makemigrations vitrines  
python manage.py sqlmigrate vitrines 0001 
python manage.py migrate

ou

python manage.py makemigrations
python manage.py migrate

Usar NGROK:

Para usar o NGROK tem que adicionar o link gerado em ALLOWED_HOSTS 
no arquivo desafio/settings.py e por mudar a definição padrão tem 
que adicionar o local host também para poder funcionar, ficando algo como:

ALLOWED_HOSTS = ['2c13-177-71-45-63.sa.ngrok.io', '127.0.0.1']
